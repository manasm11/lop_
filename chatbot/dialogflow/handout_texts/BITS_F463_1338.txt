1. Course Description: The course presents an introduction to Cryptography.

2. Scope and Objective of the Course: To learn about complexity theoretic and number theoretic
background required for modern cryptography. To learn about basic tools and applications used in
modern cryptography. To learn about some cryptographic protocols.
